# Data Analytics Project

## Overview
This project demonstrates an end-to-end Data Analytics workflow using Python, SQL, and Power BI. The project covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard development, and business reporting.

The goal of this project is to transform raw data into meaningful insights and present them through interactive dashboards and professional reports.

## How to Run the Project

### Step 1: Install Required Libraries
```bash
pip install pandas numpy matplotlib seaborn
```

### Step 2: Open Jupyter Notebook
```bash
jupyter notebook
```

### Step 3: Run SQL Queries
- Import dataset into MySQL / PostgreSQL / SQL Server
- Execute SQL scripts from `sql_queries/`

### Step 4: Open Power BI Dashboard
- Open `.pbix` file in Power BI Desktop
